# Flujo Basico Practica 2
# ¿Qué es una Base de Datos Relacional?
Es un sistema que permite almacenar y organizar información de forma estructurada.
Los datos se distribuyen en tablas que se relacionan entre sí mediante claves.
Este modelo facilita la búsqueda, actualización y manejo de grandes volúmenes de información.
Además, garantiza la integridad, consistencia y seguridad de los datos.

# ¿Qué es una Tabla en el contexto de Bases de Datos Relacionales?
Es la estructura básica donde se almacenan los datos de una base de datos relacional.
Está compuesta por filas y columnas que organizan la información.
Cada tabla representa una entidad del mundo real como clientes o productos.
Permite manejar los datos de manera ordenada y comprensible.

# ¿Qué es una relación en el contexto de Bases de Datos Relacionales?
Es el vínculo lógico que conecta dos o más tablas dentro de la base de datos.
Se establece mediante claves primarias y claves foráneas.
Permite asociar datos relacionados de diferentes tablas.
Ayuda a reducir la redundancia y mantener la coherencia de la información.

# ¿Qué representa una columna en el contexto de Bases de Datos Relacionales?
Representa un campo o atributo que describe una característica de la entidad.
Define el tipo de información que se almacenará en cada registro.
Cada columna tiene un nombre y un tipo de dato específico.
Permite clasificar y organizar la información dentro de la tabla.

# ¿Qué representa un renglón en el contexto de Bases de Datos Relacionales?
Representa un registro completo dentro de una tabla.
Contiene todos los valores correspondientes a las columnas.
Describe a un solo elemento o entidad específica.
Cada renglón corresponde a una entrada única de información.

# ¿Qué es un motor de base de datos relacional?
Es el software encargado de administrar y gestionar la base de datos.
Permite crear, modificar, consultar y eliminar información.
Utiliza el lenguaje SQL para ejecutar las operaciones.
También se encarga de la seguridad y control de acceso a los datos.

# ¿Cuáles son los motores de bases de datos relacionales más populares?
Son herramientas ampliamente utilizadas en sistemas informáticos.
Permiten manejar grandes volúmenes de información de manera eficiente.
Entre los más conocidos están MySQL, PostgreSQL y Oracle Database.
También destacan Microsoft SQL Server y MariaDB.

# ¿Cuál es el motor de base de datos ligero que no necesita un servidor?
SQLite es un motor de base de datos sencillo y ligero.
No requiere la instalación ni ejecución de un servidor.
Funciona almacenando los datos en un solo archivo local.
Es muy utilizado en aplicaciones móviles y proyectos pequeños.

# ¿Qué significa SQL en el contexto de bases de datos relacionales?
SQL significa Structured Query Language.
Es el lenguaje estándar utilizado para interactuar con bases de datos relacionales.
Permite crear tablas, consultar datos y realizar modificaciones.
Es compatible con la mayoría de los motores de bases de datos.

# ¿Cuáles son los elementos de un sistema de información basado en bases de datos relacionales?
Incluye la base de datos y el motor que la administra.
También incorpora el lenguaje SQL y los usuarios del sistema.
Se apoya en hardware, software y procedimientos definidos.
Todos estos elementos trabajan juntos para gestionar la información.
